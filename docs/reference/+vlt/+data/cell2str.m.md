# vlt.data.cell2str

  CELL2STR - Convert 1-D cells to a string
 
    STR = vlt.data.cell2str(THECELL)
 
  Converts a 1-D cell to a string.
 
  Example: 
    A = {'test','test2','test3'};
    str = vlt.data.cell2str(A)
 
        produced str = 
     '{ 'test','test2','test3' }'
  
 1-dim cells only, only chars and matricies
